# Space Shooter

## Components

* Moving Background
* Lighting
  * 3 Lighting sources
* Enemies
  * Asteroid
  * Enemy Ship
    * Bolt
* Scripts
  * Destruction
    * By Contact
    * By Time
    * By Boundary - anything goes out the screen is destroyed.
  * Player
    * Move
    * Shoot
    * Keep in the screen
  * Enemy
    * Move
    * Shoot
    * Maneuver (Enemy Ship only)
    * Random Rotation (Asteroid only)
  * Infinite Scrolling Background
